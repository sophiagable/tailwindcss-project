# tailwindcss-project

A project with tailwindcss

must install 

```sh
# Using npm
npm install tailwindcss
```

```sh
npm i postcss-cli

```

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To run in hot module reloading mode:

```sh
npm start
```

To create a production build:

```sh
npm run build-prod
```

## Running

```sh
node dist/bundle.js
```
